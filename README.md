# 🚗 Vehicle Catalog

A modern vehicle catalog web application built with **React + Vite**.

This project demonstrates component-based architecture, routing, state management with React Hooks, and responsive UI design.

---

## ✨ Features

- 🚘 Vehicle list page
- 📄 Detailed vehicle page
- 🖼 Image gallery with thumbnail switching
- 💰 Discount price calculation
- ⭐ Add and delete reviews
- 📱 Responsive layout

---

## 🛠 Tech Stack

- ⚛ React
- ⚡ Vite
- 🌐 React Router
- 🎨 CSS (custom styling)
- 🧠 React Hooks (useState, useMemo)
